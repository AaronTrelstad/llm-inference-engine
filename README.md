End-to-end LLM inference engine built from scratch, with a Rust core covering the transformer, PagedAttention KV cache, and a custom time-series database with LSM-tree compaction, B-tree indexing, and WAL for crash recovery. A Go HTTP backend handles request intake and GPU-aware scheduling, routing inference jobs to Rust workers via gRPC based on cache locality. Deployed on Kubernetes with Prometheus metrics and Docker containerization.
